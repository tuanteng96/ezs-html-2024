    1. Cài đặt spa ( nhân viên, sản phẩm, dịch vụ, khách hàng … )
    2. Vận hành hàng ngày ( check in, mua hàng, thanh toán, tích thẻ, tính hoa hồng .. )
    3. Trải nghiệm App nhân viên quản lý ( công, ca, lương thưởng, thao tác nghiệp vụ với ca được giao … )
    4. Trải nghiệm website ( kết nối liên thông đến phần mềm, đặt lịch, mua hàng & xem thông tin … )
    5. Trải nghiệm App khách hàng ( đăng nhập, quản lý, tương tác: mua hàng, đặt lịch, livechat … )
    6. Trải nghiệm thiết bị hỗ trợ ( ký điện tử, đánh giá dịch vụ, quét QR thanh toán tự động, tạo thanh toán tự động … )
